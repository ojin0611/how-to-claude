# how-to-claude

비개발자가 Claude를 제대로 쓰기 위한 안내서.

**웹사이트**: https://ojin0611.github.io/how-to-claude/

---

## 이 저장소에 있는 것

- `docs/` — 공개 웹사이트 소스. GitHub Pages로 배포됨.
  - `index.html` — 메인 (6장의 카드 + 구조 설명)
  - `deeper.html` — Claude Code 구조 더 깊이
  - `story-youtube.html` — 사례 하나 (익명화)
  - `story-dk.html` — 사례 둘 (익명화)
- `CLAUDE.md` — 이 프로젝트의 **프라이버시 규칙**과 글쓰기 방침
- `README.md` — 이 파일

## 어떻게 만들어지고 있나

비개발자 친구들/지인들과 Claude Code를 함께 써본 경험에서 나온 **실제 사례** 기반의 안내서입니다. 사례에 등장하는 인물·URL·회사명은 모두 익명화되어 있어요.

## 로컬에서 보기

```bash
# Python 기본 서버로 바로 열기
cd docs && python3 -m http.server 8000
# 브라우저에서 http://localhost:8000
```
