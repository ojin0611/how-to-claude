# how-to-claude

비개발자가 Claude를 제대로 쓰기 위한 안내서.

**웹사이트**: https://ojin0611.github.io/how-to-claude/

---

## 이 저장소에 있는 것

- `docs/` — 공개 웹사이트 소스. GitHub Pages로 배포됨.
  - `index.html` — **메인 가이드** (선생님용 · AI와 소통하는 5단계)
  - `reference.html` — 국어 예상문제 만들기를 위한 외부 자료·서비스·도구 모음
  - `advanced/` — 중급자용 / 이전 버전
    - `index.html` — Claude Code의 구조와 사용자 사례 6장 카드
    - `deeper.html` — Claude Code 구조 깊이 보기
    - `story-youtube.html` · `story-dk.html` — 익명화된 사용자 사례
    - `mac.html` · `samsung.html` · `asus.html` — 개별 사용자 프로파일
- `CLAUDE.md` — 이 프로젝트의 **프라이버시 규칙**과 글쓰기 방침
- `README.md` — 이 파일

## 메인 가이드의 청중

중학교 국어 강사(사교육) — AI 코딩 도구(Claude Code, Codex)로 **예상 문제집을 직접 만들고 싶은 선생님**. 코드·터미널·영문 약어 없이 5단계로 정리.

## 어떻게 만들어지고 있나

비개발자 친구들·지인들과 Claude Code를 함께 써본 경험에서 나온 **실제 사례** 기반의 안내서입니다. 사례에 등장하는 인물·URL·회사명은 모두 익명화되어 있어요.

## 로컬에서 보기

```bash
# Python 기본 서버로 바로 열기
cd docs && python3 -m http.server 8000
# 브라우저에서 http://localhost:8000
```
