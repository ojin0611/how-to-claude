국어 예상문제 만들기 — 참고자료

"선생님들이 직접 AI 도구로 예상 문제집을 만든다"는 목표에 도움이 될 외부 자료·서비스·도구를 정리했습니다. 강의 준비 전에 한 번 훑어보시고, 학생들에게는 필요할 때 골라 소개해주세요.

이 페이지에 있는 것

  • 1. 이미 비슷한 서비스가 있어요
  • 2. 교과서·출판사·작품 목록 (어디서 확인하나)
  • 3. 한글 PDF·HWP를 텍스트로 (대량 처리 도구)
  • 4. 어떤 AI 도구를 쓸까
  • 5. 저작권 — 안전하게 쓰는 선
  • 6. 권장 작업 흐름 한눈에 보기

01이미 비슷한 서비스가 있어요

"AI로 국어 예상문제"는 이미 상용 서비스가 여럿 나와 있습니다. "우리 학원이 가진 A·B·C 학교 기출 + 자습서를 묶어서 학생 맞춤 문제"는 외부 서비스가 해주지 못하는 부분이라, 직접 만드는 가치가 있어요. 시장 흐름 파악용으로 살펴보세요.

문제G (대교 디피니션)유료
moonjeg.com

선생님 계획과 가장 유사. 지문 텍스트 입력 또는 이미지 업로드만으로 객관식·서술형 문제와 문제지를 자동 생성. 출제 의도·유형까지 지정 가능.

기출로 (천재교과서)유료
gichulro.com

중학교 내신 대비용 AI 기출문제 은행. 학교별·시험별 기출과 유사문항 추천. 학교명으로 검색하는 구조가 참고할 만함.

족보닷컴 중등유료
zocbo.com

전국 학교 기출 + AI가 유사 문항을 추천. 출제 경향을 패턴화해서 변형 문제를 만드는 방식.

레서 (Lesser)유료
lesser.school

AI 국어 학습 튜터. 초·중·고 대주제별 비문학 지문 + 실시간 피드백. 학생 직접 사용 도구라, 선생님 도구라기보단 학생 추천용.

KICE_slayer_AI_Korean (오픈소스)무료
github.com/NomaDamas/KICE_slayer_AI_Korean

수능 국어를 AI가 직접 푸는 실험 프로젝트. 강의에선 안 쓰지만, "문제·지문·선택지를 어떻게 구조화하는지"의 좋은 예시.

02교과서·출판사·작품 목록 — 어디서 확인하나

"어느 학교가 어느 출판사 교과서를 쓰는지", "그 교과서에 어떤 작품이 실렸는지"의 공식 확인처입니다. 본문 자체는 다운로드되지 않습니다 (출판사 저작권). 메타데이터만 무료.

교과서정보관무료 · 공식
kotry.kr

한국검인정교과서협회 공식 사이트. 출판사·학년·과목·교육과정별로 교과서 목록 조회 가능. 저자·출판사·사용학년까지 확인됨.

공공데이터포털 — 교육부 교과서 정보무료 · 공식
data.go.kr

교육부가 공개한 교과서 메타데이터 일괄 다운로드. 엑셀로 받아서 그대로 데이터베이스화 가능. 단, 최신성은 직접 확인 필요.

교육통계서비스 — 교과서 채택율무료 · 공식
kess.kedi.re.kr

어느 출판사 교과서가 어느 학교에 채택됐는지 통계. "우리 동네 학교들은 주로 어느 출판사를 쓰는지" 파악할 때 유용.

디지털교과서 (에듀넷)교사 인증 필요
dtbook.edunet.net

교사 인증 시 일부 디지털 본문 열람. 학원 강사용으론 접근 제한 가능. 학교 선생님 협업으로만 활용 권장.

나무위키 — 수능 출제 문학작품무료 · 참고용
namu.wiki

수능용이지만 작품별 주제·작가·시대 정리가 잘 돼 있어, 중학 단원 작품의 메타데이터 시드(seed)로 유용. 정확성은 출처 확인 필수.

03한글 PDF·HWP를 텍스트로 — 대량 처리 도구

자습서가 HWP, 시험지가 PDF·이미지인 경우가 많죠. 한 번에 텍스트로 바꾸는 도구들입니다.

Claude의 PDF 직접 처리기본 기능
platform.claude.com

별도 OCR 없이 Claude가 PDF를 직접 읽음 (텍스트 + 차트 + 이미지). 한 번에 한 파일이 한계라, 수십 부는 도구 측에서 반복 처리.

kordoc (오픈소스)무료설치 필요
github.com/chrisryugj/kordoc

HWP·HWPX·PDF·XLSX·DOCX를 Markdown으로 일괄 변환. 한국어 표·공문서 형식에 특화. 자습서가 HWP이면 이게 가장 효율적.

iLovePDF OCR무료(제한)
ilovepdf.com

웹 OCR. 설치 없이 바로. 한국어 인식률 양호. 무료는 파일 크기·횟수 제한 있음.

PDF24 OCR무료
tools.pdf24.org

100% 무료, 광고 약함. 한 번에 여러 파일 처리. 보안 민감한 자료는 로컬 도구(kordoc) 권장.

04어떤 AI 도구를 쓸까

강의에서 다룰 두 도구의 특징. 선생님 환경에 맞춰 고르세요.

Claude Code (Anthropic)유료(월 $20~)
claude.com/product/claude-code

PC에 깔아두면 지정한 폴더의 파일을 통째로 읽고 새 파일을 만들어줌. 수십·수백 부 PDF를 한 번에 분류·정리하는 데 가장 적합. 채팅보다 길고 복잡한 작업에 강점.

Codex (OpenAI)유료(ChatGPT Plus 포함)
openai.com/codex

ChatGPT의 코딩·파일 처리 버전. 이미 ChatGPT를 쓰고 계시면 추가 가입 없이 사용 가능. 인터페이스가 더 익숙할 수 있음.

Claude 웹 / ChatGPT 웹무료 또는 유료
claude.ai · chatgpt.com

한 번에 몇 개 파일 정도, 가벼운 작업·검색에 적합. 수십 부 PDF를 한꺼번에 다루기엔 부적합.

05저작권 — 안전하게 쓰는 선

선생님 본인이 이 안내문을 학생·동료에게 전달할 때 참고할 가이드입니다. 법률 자문이 아니라 통상 권장 수준의 정리예요.

학원 내부용·비공개 보관까지는 ⭕

  • 시험지를 텍스트로 옮겨 본인 PC 또는 학원 폐쇄 폴더에 보관
  • 그 자료로 학생에게 인쇄 출력해서 풀게 하기
  • 본문 인용 시 작가·제목·출처 표기

외부 배포·공유는 ⚠️

  • 블로그·SNS·다른 학원과 공유 → 출판사 저작권 침해 가능
  • 시 본문 전문을 학생 자료에 그대로 옮겨 배포 → 작가 저작권 침해 가능 (사후 70년)
  • AI가 만든 문제를 유료로 판매 → 원 시·지문의 저작권 별도 정리 필요

학생에게 안내할 한 줄

"원문은 너희 교과서·자습서로 확인하고, 이 자료는 풀이용으로만 사용하자."

06권장 작업 흐름 한눈에 보기

메인 가이드의 5단계를 이 강의 상황에 맞춰 다시 정리하면 이렇습니다.

① 자료 모으기 (수업 전)

학교별 시험지(PDF/사진), 교과서·자습서를 한 폴더에 모음. 파일명 규칙(학교_연도_시험_과목.pdf) 통일.

② 텍스트로 변환 (한 번에)

Claude Code 또는 kordoc로 폴더 통째로 변환. "PDF를 전부 텍스트로, 학교별·연도별로 분류해서 표로 정리"가 한 줄 명령.

③ 작품 DB 만들기

시·소설·비문학 지문을 작가·제목·주제·표현기법·시대·수록 교과서 열로 정리(엑셀). 30~50편이면 시작 충분.

④ 문제 DB 만들기

문제 1개 = 1줄. 출제학교·연도·관련 작품·발문·보기·정답·해설·문제 유형. 유형 분류는 AI에게 부탁.

⑤ 예상문제 생성

"C중 2학년 2학기 중간, 시 단원. 학생이 보는 작품은 다·바·사. 우리 DB에서 가·나·라·마 중 주제·표현이 비슷한 것을 뽑아 묶어 N문항 생성. 형식은 이전 문제집과 동일."

⑥ 검수 (반드시)

생성된 문제는 처음엔 100% 선생님이 직접 풀어 정답·해설 확인. 자주 틀리는 패턴은 다음 부탁 시 "이런 실수는 피해줘"로 미리 차단.
