저평가 매물을 빠르게 찾는 리셀 사업

ASUS · Windows Claude Pro 페르소나 · 자동화

지금 하고 싶은 것

중고차 시장에 존재하는 정보 비대칭과 급매를 이용해, 실제 가치보다 낮게 올라온 매물을 빠르게 발견하고 적정 시세로 되파는 차익거래 기반 리셀 사업을 만들고 싶다. 주식시장의 가치투자자가 저평가 종목을 찾듯이.

풀고 싶은 문제들

딜러망과 공개 플랫폼 사이의 가격 격차를 찾아내기

같은 차종·같은 연식·비슷한 상태인데 채널마다 가격이 다릅니다. 그 차이를 데이터로 잡아 어디서 사서 어디에 팔면 유리한지를 자동으로 보여주는 눈.

겉으로 드러나지 않는 가치를 점수화하기

외장·내장 색상 조합, 선호도 높은 옵션, 관리 상태 같은 정성적 요소가 실제 재판매 가격을 크게 좌우합니다. 이걸 주관이 아니라 근거 있는 점수로 환산.

급매물을 실시간으로 걸러내기

시세 대비 크게 아래로 이탈한 매물을 실시간으로 스캔해서 알림. 사람이 수시로 플랫폼을 들여다보지 않아도 되게.

AI가 특히 조심해야 하는 부분

카드 여섯 — 조용한 실패 + 카드 둘 — 마당

돈이 바로 오가는 분야이기 때문에 위험이 두 갈래입니다. 하나는 AI가 시세를 지어내거나 낡은 데이터로 판단하는 것 — 출처와 기준일을 항상 남겨야 합니다. 다른 하나는 AI가 실제 매수·결제까지 실행하게 두는 것 — 분석·추천은 AI, 매매 결정과 실행은 사람이라는 경계를 처음부터 못박아야 합니다.
