질문 네 개

  1. 요즘 반복해서 하는 일 중에 "이거 누가 대신 좀…" 싶은 게 있나요?
  2. 완전히 새로 만들고 싶은 것(프로젝트·콘텐츠·사업)이 있나요?
  3. 그걸 하는데 지금 막힌 건 정보 부족, 시간 부족, 용기 부족 중 어느 쪽에 가깝나요?
  4. 만약 AI가 한 가지만 잘해준다면, 뭐를 맡기고 싶으세요?

대답이 한 줄이어도 좋아요. 여기서 페르소나(창작자 / 분석가 / 자동화)와 먼저 읽을 카드가 정해집니다.
